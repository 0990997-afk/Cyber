# Funnel map — від розгубленості до продажу

Воронка показує, як AI-сомельє веде гостя від «нічого не розумію у вині» до впевненої (і дорожчої) покупки. Та сама воронка візуалізована на лендингу (секція «Як це продає»).

```
┌──────────────────────┐
│ 1. Розгублений        │  Гість біля полиці / на сайті.
│    покупець           │  Емоція: тривога, «візьму найдешевше».
└──────────┬───────────┘
           │  тригер: блок «Не знаєте, що взяти? Спитайте сомельє»
           ▼
┌──────────────────────┐
│ 2. Опис вечері        │  Пара слів («стейк на вечерю») АБО фото страви.
│    або фото           │  Бар'єр майже нульовий — не треба знати сорти.
└──────────┬───────────┘
           │  AI міркує: їжа + бюджет + каталог магазину
           ▼
┌──────────────────────┐
│ 3. Три варіанти       │  Бюджетно · Золота середина · Здивувати гостей.
│    з поясненням       │  Кожне «чому саме це» + ідея закуски.
└──────────┬───────────┘
           │  делікатний апсейл: видно діапазон, не лише ціну
           ▼
┌──────────────────────┐
│ 4. Купити /           │  Один клік. Рекомендація прив'язана до
│    забронювати        │  реального інвентаря з ціною.
└──────────┬───────────┘
           ▼
┌──────────────────────┐
│ 5. Продаж і           │  Вищий чек + гість, який повертається,
│    повернення         │  бо вино «зайшло» і йому пояснили чому.
└──────────────────────┘
```

## Що рухає конверсію на кожному кроці
| Крок | Що знижує тертя | Що піднімає чек |
|---|---|---|
| 1 → 2 | Помітний CTA, нульовий поріг входу (опиши/сфоткай) | — |
| 2 → 3 | Швидка відповідь, людська мова без жаргону | Три рівні замість однієї «дешевої» опції |
| 3 → 4 | Конкретика «чому це підходить» знімає страх помилки | Видимий діапазон → апсейл до «золотої середини» |
| 4 → 5 | Кнопка «купити/забронювати» поряд із порадою | Закуска-допродаж, привід повернутися |

## Метрики для пілота
- **CR кроку 2→3** (дійшли до рекомендації) — головний показник залученості.
- **Mix цінових рівнів** (скільки обирають middle/premium) — прямий доказ апсейлу.
- **Δ середнього чека** vs. сесії без сомельє.
- **Повернення / повторні запити** — лояльність.
